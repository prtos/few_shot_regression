\subsubsection{Abstract}\label{abstract}}

In this experiement, we focus on comparing different ways of obtaining
support set representations. We also compare MetakRR with Conditional
neural process (CNP) and Bayesian MAML (BMAML, not finished yet). We
only use the harmonic meta-dataset for this experiment. Recall that this
dataset consists of two sin functions added to generate a harmonic
function. A function gives a learning task and all tasks differ from
each others by the phases and amplitudes and the frequences of the sin
functions combined.

To compute the support set representation, we tested the following
architectures: - deepset, a MLP based network with order invariance
proposed by https://papers.nips.cc/paper/6931-deep-sets.pdf - set2set,
an lstm based network proposed by https://arxiv.org/abs/1511.06391 -
multihead attention (from attention is all you need) - simple attention
(from neural machine translation) - relation net with average pooling -
relation net with attention pooling - relation net with deepset pooling

Once the support set representation are computed, we use the fusion
network to compute the conditionned input representation: + fusion net
\(f([\phi(x); z])\): where \([a; b]\) is the concatenation of the
vectors \(a\) and \(b\); and \(f\) is neural network
